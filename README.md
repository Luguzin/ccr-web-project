# CCR Web Project

Projeto de página institucional inspirado no Grupo CCR, com foco em mobilidade humana e sustentável.

## Acesse o site

[![Website](https://img.shields.io/badge/Website-ByteCraft%20Solutions-blue?style=for-the-badge )]([https://luguzin.github.io/ccr-web-project/] ) 

##  Sobre o projeto

Este projeto consiste em uma página web estática desenvolvida com HTML e CSS, apresentando informações sobre a CCR, sua atuação, missão, valores, negócios e desafios no setor de infraestrutura.

##  Objetivo

Criar uma interface moderna, responsiva e informativa, aplicando conceitos de:
- Estruturação semântica com HTML5
- Estilização com CSS3
- Layout responsivo (Flexbox e Grid)
- Design institucional

##  Tecnologias utilizadas

- HTML5
- CSS3
